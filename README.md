# About

## Folder structure

```json
src/
├── assets/ - Stores static assets like fonts, images, and potentially global styles.
├── components/ - React components categorized by complexity
│   ├── atoms/ - Basic building blocks (buttons, inputs, etc.).
│   ├── molecules/ - Groups of atoms (forms, navigation bars, etc.).
│   └── organisms/ - Reusable UI components (cards, modals, etc.).
├── contexts/ - For managing global state with a Context API.
├── service/ - Api interaction.
├── hooks/ - hook call api using service..
├── pages/ - Pages separated by modules.
└── utils/ - Functions reusable.
```
